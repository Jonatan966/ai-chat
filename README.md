# AI Chat - Experimento

Este projeto consiste em um simples chatbot utilizando a API da OpenAI e a Vercel AI SDK.

### Como iniciar

Para iniciar a aplicação, é necessário configurar a variável de ambiente `OPENAI_API_KEY`.
